# Day4 - Swapping of 2 numbers
- Get 2 numbers as the input 
- print input
- Swap those 2 number 
- print swaped value
---
## TestCase 1:-
---
### Input:
```
Enter the number 1: 98
Enter the number 2: 43
```
### Output:
```
Before Swap : 
Number 1 = 98
Number 2 = 43
After Swap : 
Number 1 = 43
Number 2 = 98
```
